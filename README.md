
# Viralizoou

Viralizoou é uma ferramenta com inteligência artificial que te ajuda a criar produções cinematográficas detalhadas a partir de uma simples ideia. Atuando como diretor e roteirista, a aplicação gera tudo o que você precisa para dar vida à sua visão.

## Funcionalidades

- **Geração Detalhada de Roteiros:** Transforma seu conceito inicial em um roteiro completo, com ações e diálogos.
- **Entrada Multimodal:** Aceita suas ideias em diversos formatos:
    - Descrições em texto
    - Imagens de referência
    - Clipes de vídeo
    - Inspiração em áudio
- **Saída Abrangente:** Gera um pacote de produção completo, incluindo:
    - **Sinopse:** Um resumo impactante da história.
    - **Roteiro Detalhado:** Com cenas, ações e diálogos.
    - **Descrições de Cena:** Focadas em cenário, iluminação e ângulos de câmera.
    - **Perfil Visual:** Detalhes únicos para garantir a consistência de personagens e ambientes.
- **Geração de Referência Visual:** Cria referências visuais para as cenas com base no roteiro.

## Como Funciona

A aplicação utiliza modelos de IA avançados para interpretar sua entrada e gerar um roteiro cinematográfico rico e estruturado. O núcleo da aplicação é construído com Genkit e Next.js.

## Começando

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/adrielmoraes/Viralizoou.git
   ```
2. **Instale as dependências:**
   ```bash
   npm install
   ```
3. **Execute o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
